# PetHome.lk – Full-Stack Pet Adoption Platform

PetHome.lk connects pet adopters with shelters, providing a secure platform for adopting and managing pets. Admin controls ensure smooth operations while users can browse and adopt pets safely.

---

## Technologies Used
- React.js, Node.js, Express.js  
- MongoDB, Tailwind CSS  
- RESTful APIs  

---

## Features
- Role-based authentication (Admin & User)  
- Admin panel for managing pets and users  
- Browse and filter pets easily  
- RESTful APIs for efficient data retrieval  
- Responsive design for mobile and desktop
